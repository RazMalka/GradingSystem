### Prerequisites and Libraries
- ReactJS
- ExpressJS
- NPM
- Yarn

### Installing and Running
- Clone the Project
```
git clone https://github.com/RazMalka/GradingSystem.git
cd GradingSystem
```
- Install and Execute Client
```
cd client
yarn install
yarn start
```
- Install and Execute Server
```
cd server
npm install
npm start
```
